# Kit: An extremely simple text file viewer
This is a really simple text viewer written in C, you can only read text files. That's it ;-;  

**If there is an issue please [report](https://github.com/kittyteggie/kit/issues) it :)**  

# Compiling
Compiling is extremely simple, just use gcc, a command like this:  
`gcc main.c -o kit`
  
And then just run it:  
`./kit file.txt`
